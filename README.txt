Géonigme

Site internet communautaire pour créer et jouer à des chasses aux trésors dans la ville de Montpellier.

http://geonigme.fr/

Répertoires :
	- app : code de l'application
		- assets : les fichiers .less (.css) et .coffee (.javascripts)
		- controllers : les Contrôleurs
		- forms : les classes décrivant des formulaires
		- global : classes globales (gérant l'application Play, et Sésame)
		- helpers : fonctions d'aide pour les templates
		- models : classes gérant la logique de l'application
		- views : templates (génère les vues)

	- conf : Fichiers de configurations (routes et autres)

	- conversion-interconnexion : Tous les fichiers récupérés/générés de données provenant du web

	- présentation : Fichiers rapport pdf/powerpoint

	- public : Librairies css/javascripts
